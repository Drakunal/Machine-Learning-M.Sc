## Run Locally

Clone the project

```bash
  git clone https://github.com/Drakunal/Machine-Learning-M.Sc.git
```

Go to the project directory

```bash
  cd Machine-Learning-M.Sc
```

Opening in VS Code

```bash
  code .
```

Opening in Jupyter notebook

```bash
  jupyter notebook
```
